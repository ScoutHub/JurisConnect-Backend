Login

/auth/login

+

Login user to api

Path: /auth/login

Headers

{
  Content-Type: application/json
}

Body

{
  string email,
  string password
}

Register

/auth/register

+

Register user to api

Path: /auth/register

Headers

{
  Content-Type: application/json
}

Body

{
  string email,
  string username,
  string first_name,
  string last_name,
  string confirm_password
}

Get refresh token

/auth/refresh/token

+

Get new access and refresh token

Path: /auth/refresh/token

Headers

{
  X-REFRESH-TOKEN: refresh_token
}
